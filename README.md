# 🥚 Egg Timer

Pick how you want your egg, walk away, come back when it beeps.

## Stack

React + TypeScript, React Router, Bootstrap 5.

## Setup

```bash
git clone https://github.com/itj-7/EggTimer.git
cd egg-timer
npm install
npm run dev
```

Drop `Alarm.mp3` and the egg images in `public/` before running.

## Cooking times

| Type       | Time   |
| ---------- | ------ |
| Soft       | 3 min  |
| Medium     | 6 min  |
| Hard       | 8 min  |
| Extra Hard | 10 min |
